\doxysubsection{Detailed Description}
Interface para apresentação de investimentos e pagamentos. 

Esta classe fornece uma interface para o módulo de apresentação de investimentos (MAI), que valida o formato dos dados e solicita serviços do módulo de serviço de investimentos (MSI) para realizar operações CRUD.
